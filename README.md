morseLedPi
==========

Give it a word, it blink your LED in morse code.

Requirement
===========
It requires [WiringPi](http://wiringpi.com/download-and-install/) 
And a RaspberryPi 

Example
===========

./morseTest 1 sos

"1" : the pin to use [see mapping](http://wiringpi.com/pins/)

"sos" : the word to convert in morse code
